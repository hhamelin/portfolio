An interesting project, that evolved with me. When I first started it, I was learning this (at the time) new game engine, "Godot". I had a fun time learning the in's and out's of its UI systems. But at some point, I decided I wanted to move it on to the web, where it would be more useful. Godot at that time was not well-equipped to handle web exports, so I decided to rewrite the code in JavaScript. This gave me a chance to dust off my JavaScript skills, and find out how to create this kind of reactive embeddable application in vanilla JS.

Alchemy Calculator

Live Demo Source Code

A recreation of an "alchemy crafting" system that uses bitwise math on the item's unique code. This code is represented as a punched card, which can be combined with other cards through various operations. The original had ‘and’ and ‘or’, but I also added ‘xor’ and ‘abjunction’.

About

My name is Haven Hamelin, and I am currently pursuing a degree in Game Design at IU Indianapolis. I am passionate about creating engaging digital experiences and have been working with computers for as long as I can remember. My skill set includes strong proficiency in various programming languages such as Java, JavaScript, Python, C#, and GDscript (for the Godot game engine).

I am also experienced with Linux and have developed a growing talent for digital art, particularly pixel art, along with an emerging interest in music composition.
I pride myself on being a dedicated and focused individual. When I am passionate about something, I am committed to seeing it through to its fullest potential. I believe strongly in the importance of investing time in developing one's skills and passions, and I aim to build a career that allows me to do exactly that. My drive to continuously learn and create fuels my ambition to contribute to the game design industry.
