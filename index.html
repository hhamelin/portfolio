An interesting project, that evolved with me. When I first started it, I was learning this (at the time) new game engine, "Godot". I had a fun time learning the in's and out's of its UI systems. But at some point, I decided I wanted to move it on to the web, where it would be more useful. Godot at that time was not well-equipped to handle web exports, so I decided to rewrite the code in JavaScript. This gave me a chance to dust off my JavaScript skills, and find out how to create this kind of reactive embeddable application in vanilla JS.

Alchemy Calculator

A recreation of the alchemy crafting system featured in a popular piece of online media. It involves a series of complex math operations that involve a “Captchalogue Code”, used as a unique identifier. This code is then represented as a punch card, which can be combined with other punch cards through bitwise operations. The original only had ‘and’ and ‘or’, but I ended up adding ‘xor’ and ‘abjunction’.

Tools Used: HTML, JS, Aseprite
Live Demo: Alchemy Crafting

Jester Jabs

A multiplayer party game themed around jesters, created with a team for a game jam. It consists of a variety of minigames in which you attempt to best the other players through fun challenges. The minigames are joke-based, like creating the punchline for a set-up, or visa versa. This ended up with a familiar, yet distinct experience to popular party games like Jackbox.

Tools Used: Godot, Aseprite
Live Demo: Jester Jabs
Source Code: GitHub

Shopkeep Keepaway

A fun arcade-style alt.ctrl game, where the player has to fulfill their “shopping” list without catching the attention of the shopkeep. Move a ruler back and forth (or a mouse if that’s all you’ve got) to take your aim in this wacky game, Shopkeep Keepaway!

Snatching Sorcerers

A two player infinite arcade-style action game, in which one player controls the character on the top with the mouse and the other player controls the bottom player using the arrow keys. The top player has to do their best to catch the elements falling from the top of the screen, and if they fail then the elements will turn into enemies for the bottom player to avoid.

Tools Used: Godot
Live Demo: Shopkeep Keepaway
Source Code: GitHub

Voxel Sandbox

A 3D voxel-based sandbox simulation, similar in functionality to the game Minecraft. The goal was not to create a fully featured game experience, but instead to illustrate my ability to create such a game within the Godot game engine. Kickstarted by the discovery of an open-source voxel engine that I could implement within the project.

Tools Used: Godot
